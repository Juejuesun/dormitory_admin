# 适友阁控制台
软工实践大作业———适友阁admin部分
<br>
<br>

## 本地运行

```shell
# 克隆项目
git clone -b master https://github.com/Juejuesun/dormitory_admin.git
# 进入项目目录
cd dormitory_admin
# 安装依赖
npm i
# 本地开发 启动项目
npm run serve
```
